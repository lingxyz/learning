## 快速开发

```bash
# 安装依赖
npm i docsify-cli -g

# 实时编译预览
docsify serve .
```

## 目录说明
```bash
|-- app
|   |-- docs    # 文档
|   |-- assets  # 图片资源
|   |-- plugins # docsify 自定义插件
|   |-- styles  # docsify 重置样式
```
